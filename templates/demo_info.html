Full Version Setup

To enable full features with Supabase:
  1. Create Supabase Project:
    • Go to supabase.com
    • Create a new project
    • Copy your project URL and anon key
  2. Setup Database:
    • Run the SQL schema from backend/database_schema.sql
    • This creates tables for calculations and location data
  3. Configure Environment:
    • Update .env file with your Supabase credentials
    • Add Global Solar Atlas API key (optional)
  4. Run Full Application:
    • Use python solar_app.py instead of demo_app.py
    • All features including PDF reports will be available
